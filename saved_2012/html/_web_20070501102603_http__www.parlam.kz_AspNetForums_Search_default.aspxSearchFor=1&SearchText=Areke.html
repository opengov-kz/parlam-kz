1.Письмо Депутату Утепбаеву Б.А.
Отправлено Areke on 13.04.2007 11:21:28
Здравствуйте! Жителями г.Щучинска было выслано письмо депутату Утепбаеву с просьбой защитить права владельцев праворульных автомобилей. Просим подтвердить получение письма. Во время приема избирателей в г.Щучинск депутату была подана жалоба от Абдрахмановой К.М. по поводу отказа районного акимата в предоставлении прав на земельный участок в пос.Боровое. Данную жалобу можно не рассматривать, так как этот вопрос удалось решить в районном суде Щучинска. Просим передать данное сообщение депутату ...
