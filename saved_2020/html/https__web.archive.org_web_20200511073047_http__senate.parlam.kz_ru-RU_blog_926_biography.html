19 марта 2019 года Указом Президента Республики Казахстан Н.А.Назарбаева назначен – Депутатом Сената Парламента Республики Казахстан

В 1983 году окончил Республиканскую физико-математическую школу-интернат Алматы.

В 1987 году окончил Ленинградский государственный университет имени А.Жданова. Юрист.

С 1987 по 1996 годы служба в органах прокуратуры

С 1996 по 1999 годы служба в таможенных органах

С 1999 по 2001 годы служба в органах налоговой полиции

С 2001 по 2012 годы служба в органах финансовой полиции

С 2004 по 2007 год — Начальник Департамента по борьбе с экономической и коррупционной преступностью (финансовая полиция) по Актюбинской области.

С 2007 по 2008 год — Заместитель Председателя Агентства Республики Казахстан по борьбе с экономической и коррупционной преступностью (финансовая полиция).

С 2008 по 2012 год — Председатель Агентства Республики Казахстан по борьбе с экономической и коррупционной преступностью.

Генерал-майор финансовой полиции.

2012 по 2013 год — Аким Акмолинской области.

2013 по 2014 год — Помощник Президента Республики Казахстан — Секретарь Совета Безопасности Республики Казахстан.

2014 по 2015 год — Председатель Агентства Республики Казахстан по делам государственной службы и противодействию коррупции.

2015 по 2017 год — Председатель Национального бюро по противодействию коррупции Министерства по делам государственной службы Республики Казахстан.

2017 по 2019 год – Генеральный прокурор Республики Казахстан.

Государственные и ведомственные награды:

Орден «Данқ» I степени (2014 г.)

Орден «Данқ» II степени (2006 г.)

Юбилейные Медали

Ведомственные медали
